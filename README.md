Movies App

This android app displays a list of the current movies that are playing in theaters. The app retrieves data from an API 
and presents a description along with the name and poster. 



Technology Used:
A variety of technology and languages were used for this app such as Kotlin, Glide, and TMDb API

The app will send a request to the api, which then the JSON takes the response and turns it into objects.
The xml files creates the format of which each movie is displayed.
